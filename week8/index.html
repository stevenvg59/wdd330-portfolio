TRANSFORM

With the "transform" property we can translate, rotate, scale, and/or skew any element on the page. There are transform functions such as translations, scaling, rotation, and skew that help on that approach. You can add all these functions inside the transform property line, like so:

transform: rotate(10deg) translateX(40px) scale(1.5) skew(15deg, 4deg);

TRANSITION

Transitions allow the values of CSS properties to change over time, which could also be considered as a simple animation. They have some properties such as transition-property, transition-duration, transition-timing-function, and transition-delay. For example this is the syntax for these properties:

transition-property: transform;
transition-duration: 0.2s;
transition-timing-function: ease-out;
transition-delay: 50ms;

When we combine transitions with transforms we can get a more dynamic page.

  • Using the Canvas API we can draw anything we want only using Javascript. With canvas, we can draw shapes and lines, arcs and text, gradients and patterns.
  • SVG stands for Scalable Vector Graphics, a specific file format that allows you to describe vector graphics using XML. Unlike canvas, images created with SVG are available via the DOM.
  • The "Drag and Drop" API allows us to specify that certain elements are draggable, and then specify what should happen when these draggable elements are dragged over or dropped onto other elements on the page.
